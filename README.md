# 📋 Java To-Do Application

A sleek, desktop-based To-Do list manager built with Java and Swing. 

## ✨ Features
- **Clean UI**: A beautifully customized aesthetic with a translucent search bar and light blue accent buttons.
- **Interactive Tasks**: Click on any task to instantly cross it off your list. Tasks support visual checkbox tracking (☐ and ☒).
- **Persistent Storage**: Your tasks are automatically saved locally, ensuring you never lose your progress between sessions.
- **Task Management**: Easily add, edit, delete, and clear completed tasks.
- **Dynamic Status Bar**: Real-time tracking of Total, Completed, and Pending tasks.
- **Hover Status**: Quick tooltip checks for pending or completed tasks.

## 🚀 Getting Started

### Prerequisites
- **Java Development Kit (JDK) 8** or higher.

### Compilation and Execution
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/Ismail-Maalim/To-do-application-.git
   cd "to-do app"
   ```
2. Compile the source code located in the `src` directory:
   ```bash
   javac -encoding UTF-8 src/TodoApp.java
   ```
3. Run the desktop application:
   ```bash
   java -cp src TodoApp
   ```

## 🛠️ Built With
